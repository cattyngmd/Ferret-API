# Color

## Methods

### color(r, g, b, a): Color

| Type | name  |
| ---- | ----- |
| int  | red   |
| int  | green |
| int  | blue  |
| int  | alpha |

```lua
local color = color(255, 0, 255, 255)
```

### color(r, g, b, a): Color

| Type   | name  |
| ------ | ----- |
| double | red   |
| double | green |
| double | blue  |
| double | alpha |

```lua
local color = color(1.0, 0.0, 1.0, 0.5)
```
