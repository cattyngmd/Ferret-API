## Loading Lua scripts <a href="#loading-lua-scripts" id="loading-lua-scripts"></a>

1. Drop the script into the `.minecraft\ferret\scripts` folder.
2. Open a chat in minecraft.
3. Type `$lua load <script_name>`.
